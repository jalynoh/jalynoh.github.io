Signal Way

ongoing

About the Project

A web-based tool that will help with some of the work I do at UTAPD, shifting physical paper alarm testing to digital. The process of alarm testing on paper is tedious at UTA as there are a lot of checks that need to be performed. There is currently know viable solution, so I decided to develop my own using the Python Flask Web Framework. When finished, this project should cut down analysis time and be able to give a visual representation on the state of campus alarms.
